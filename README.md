# ☀️ SunnyTasks

A beautiful Django-based To-Do application with user authentication, task management, and a clean pastel-themed UI.

## Features

- User Registration
- User Login & Logout
- Create Tasks
- Mark Tasks as Complete
- Delete Tasks
- User-specific Task Lists
- Responsive UI using Bootstrap

## Installation

```bash
git clone https://github.com/IshikaGarg787/SunnyTasks
cd SunnyTasks

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver

Visit:
http://127.0.0.1:8000/
