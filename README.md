# ☀️ SunnyTasks

A modern Django-based To-Do web application with secure user authentication, personalized task management, and a beautiful pastel-inspired UI.

SunnyTasks helps users organize daily activities efficiently with a clean and responsive interface built using Django and Bootstrap.

---

## ✨ Features

* 🔐 User Registration & Authentication
* 👤 Secure Login & Logout System
* ✅ Create, Update & Manage Tasks
* ✔️ Mark Tasks as Completed
* 🗑️ Delete Tasks
* 📋 User-Specific Task Dashboard
* 🎨 Beautiful Pastel-Themed UI
* 📱 Responsive Design using Bootstrap

---

## 🛠️ Tech Stack

* Python 🐍
* Django
* HTML5 & CSS3
* Bootstrap
* SQLite

---

## 📂 Installation & Setup

### Clone Repository

```bash
git clone https://github.com/IshikaGarg787/SunnyTasks
cd SunnyTasks
```

---

### Create Virtual Environment

```bash
python -m venv venv
```

---

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

### Apply Migrations

```bash
python manage.py migrate
```

---

### Run Development Server

```bash
python manage.py runserver
```

---
